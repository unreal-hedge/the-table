# The Table ♠

Private poker for the boys. Phase 1a: No-Limit Hold'em cash game, hot-seat on one screen.

## Run it

    npm install     # poker-ts patch auto-applies
    npm run dev     # http://localhost:3000

## Test the engine (do this before every push)

    npx tsx test-engine.ts

## Deploy

Push to GitHub → import the repo on vercel.com → deploy. No env vars needed for 1a.
